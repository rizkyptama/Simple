      作用機轉：

      Tadalafil 為具選擇性、可逆性 c-GMP 之phosphodiesterase type 5 (PDE5)抑制劑。當性刺激造成一氧化氮局部釋出時，tadalafil 可抑制 PDE5 作用而提高 c-GMP 於海綿體濃度使平滑肌放鬆和血流流入陰莖而勃起。由於PDE5 抑制 c-GMP 濃度的作用亦出現於攝護腺平滑肌、膀胱及其血管系統中，tadalafil 可用以減緩良性攝護腺(前列腺)肥大症所伴隨的下泌尿道症狀。

      使用禁忌：

      禁用於規律或間歇服用任何形式 nitrates類藥物、不適合性行為之心臟病患、因非動脈性前部缺血性視神經病變而一眼失明或已知對 tadalafil 過敏之患者。

      不良反應：

      常見副作用為頭痛、消化不良及腰背痛。

      注意事項：

      1. 本品為膜衣錠，不可剝半使用，應整顆服用。
      2. 本品不適用於女性及孩童(<18 歲) 。
      3. 嚴重肝臟功能不全(Child Pugh Class C)及腎功能不全(肌酸酐清除率＜30ml/min)或需血液透析者，不建議每日使用。
      4. 對於每日服用此藥品之患者，需審慎評估與其他藥物 nitrates、alpha 阻斷劑、CYP3A4 抑制劑(如 ritonavir、ketoconazole及 itraconazole)之交互作用。

      藥品特色：

      1. 本品服用後，作用時間可長逹 36 hrs。
      2. 本品可有效改善良性攝護腺肥大患者之前列腺症狀評分(IPSS)及餘尿量，有效治療相關下泌尿道症狀。
      3. 對於曾使用需要時服用 tadalafil 10mg 或20mg 有效，且需經常服用（每週至少二次）之患者，基於醫師專業判斷及患者意願，可考慮每天一次低劑量 tadalafil 5mg治療，療效較佳，副作用與需要時服用相當。

      安全性評估

      目前相關的研究結果並不多。已知的結果指出，使用低劑量犀利士每日錠和依需求使用（on demand）來比較，副作用的種類和發生比率並沒有多大差異。其中最常被報告出來的副作用是消化不良，頭痛，背痛和疑似感冒的症狀。

      絕對禁忌

      和所有PDE5-I一樣，犀利士禁用於使用任何形式有機硝酸鹽的患者。若需給予硝酸鹽藥物，必須間隔至少48小時，並由醫護人員藉由適當的血液動力學監測，進行嚴密監控。另外，犀利士也禁用於因非動脈性前部缺血性視神經病變（NAION）而失去單眼視力的患者。
